Detailed Description

A driver for reading from Quadrature Encoders.

This file creates an encoder calss that contains 3 methods that can be used in other files to get the change of the position of an encoder over time. It contains a construcotr method to instantiate each class object. A get position method that returns the position of the encoder. A zero poistion method that zeroes the position of the encoder to a specifed value.
