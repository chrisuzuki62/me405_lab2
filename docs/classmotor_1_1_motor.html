Detailed Description

Interface with ME405 motors.

Creates a class that can be called into other python files that is used to interface and set the duty cycle of a motor. The class contains methods to enable and disable the motors and to set the duty cycle.

Constructs a motor object.

Creates a motor driver by initializing GPIO pins and turning the motor off initially for safety. This also sets up the pin objects for motors 1 and 2 for use in other files.

Parameters
motor_numAn integer to describe each instance of the Motor object. There are a unique set of pins for each instance of the Motor. Currently, the driver supports up to two motors.
